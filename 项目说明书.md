# 清璃俱乐部官方网站 - 项目说明书

## 项目简介

清璃俱乐部官方网站是清璃俱乐部技术人员开发的一个展示清璃俱乐部陪伴服务的单页网站，主要提供王者荣耀、和平精英、蛋仔等热门游戏的陪玩及代打服务和陪聊服务展示，包含价格表、团队介绍、纳新公告等核心功能模块。网站采用现代化设计风格，适配移动端和桌面端，旨在为用户提供清晰的服务信息和良好的浏览体验。

## 功能特点

- 响应式设计，适配各种设备屏幕
- 服务项目与价格透明展示
- 团队介绍与门面展示
- 纳新公告信息发布
- 联系咨询入口
- 平滑滚动与交互动画
- 欢迎弹窗引导访问

## 技术栈

- HTML5 语义化标签
- Tailwind CSS v3 样式框架
- Font Awesome 图标库
- JavaScript 交互逻辑
- Chart.js (引入未使用，预留数据可视化)
- 自定义动画与过渡效果

## 安装与使用

1. 克隆或下载项目文件到本地
2. 直接打开 `index.html` 文件即可本地运行
3. 建议使用 Edge 浏览器访问以获得最佳体验（避免部分浏览器兼容性问题）

## 主要模块说明

1. **首页（Home）**：展示俱乐部核心信息与服务亮点
2. **关于清璃（About）**：介绍俱乐部背景、理念与团队优势
3. **价格表（Prices）**：详细展示各类游戏陪玩与代打服务价格
4. **门面展示（Team）**：展示团队成员信息（待完善）
5. **纳新公告（Recruit）**：发布招募信息（待完善）
6. **联系我们（Contact）**：提供咨询与联系入口（待完善）

## 注意事项

- PC端用户建议使用 Edge 浏览器访问，避免部分内容错位
- 网站图片资源需放置在 `图片文件` 目录下，确保路径正确
- 若需修改样式，可通过 Tailwind 配置或自定义 CSS 进行调整
- 价格信息和服务内容可根据实际业务需求在对应 section 中修改
